## Installation

To get started with ClyCites, follow the steps below:

1. Clone the repository:

   ```bash
   git clone https://github.com/ClyCites.git
   ```

2. Navigate into the project directory:

   ```bash
   cd ClyCites
   ```

3. Install the required dependencies:

   ```bash
   npm install
   # or for Python:
   pip install -r requirements.txt
   ```

4. Run the project:

   ```bash
   npm start
   # or for Python:
   python app.py
   ```
